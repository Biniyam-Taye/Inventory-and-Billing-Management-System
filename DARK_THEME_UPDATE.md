# Dark Theme Authentication - Design Update

## Overview
Updated the login and register screens to match the dark theme of your main inventory application.

## Design Changes

### Color Scheme
**From:** Colorful gradients (purple, pink-red) with glassmorphism
**To:** Dark professional theme matching the main app

#### New Color Palette:
- **Background:** `#1a1a1a` (Very dark gray)
- **Card Background:** `#2a2a2a` (Dark gray)
- **Card Border:** `#3a3a3a` (Medium gray)
- **Primary Accent:** `#ff9800` (Orange) - Used for titles and links
- **Button Color:** `#00bcd4` (Cyan/Turquoise) - Matching app buttons
- **Text Color:** `#e0e0e0` (Light gray)
- **Secondary Text:** `#b4b4b4` (Medium gray)
- **Input Background:** `#1a1a1a` (Very dark)
- **Input Border:** `#3a3a3a` (Medium gray)

### Visual Elements

#### Login Screen
- **Title:** "Inventory Management" in orange (#ff9800)
- **Subtitle:** "Sign in to continue" in light gray
- **Input Fields:** Dark background with subtle gray borders
- **Button:** Cyan "Sign In" button with hover effects
- **Link:** Orange "Sign Up" link with hover animation

#### Register Screen
- **Title:** "Create Account" in orange (#ff9800)
- **Subtitle:** "Join us today" in light gray
- **Input Fields:** 5 fields (Full Name, Username, Email, Password, Confirm Password)
- **Button:** Cyan "Create Account" button
- **Link:** Orange "Sign In" link with hover animation
